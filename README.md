# wsasdsda.github.io

这是一个可直接部署到 GitHub Pages 用户主页仓库的静态个人主页模板。

部署目标：

- 仓库名：`wsasdsda.github.io`
- 发布网址：`https://wsasdsda.github.io/`

## 文件说明

- `index.html`：主页内容
- `assets/styles.css`：页面样式
- `.nojekyll`：关闭 GitHub Pages 的 Jekyll 处理

## 部署步骤

1. 在 GitHub 上创建仓库 `wsasdsda.github.io`
2. 把当前目录里的文件上传到该仓库根目录
3. 进入 GitHub 仓库 `Settings > Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/ (root)`
6. 保存后等待几分钟
7. 打开 `https://wsasdsda.github.io/`

## 发布前需要替换的内容

请编辑 `index.html` 中这些占位信息：

- `Your Name`
- 邮箱地址
- 单位与城市
- 个人简介
- 研究方向
- 论文、项目、经历、荣誉
- 外部链接

## 本地预览

直接双击 `index.html` 即可预览。
